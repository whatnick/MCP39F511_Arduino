# MCP39F511 Arduino
Arduino driver for MCP39F511 Energy Monitor ASIC
This repository aims to put together a library with
- Multi-host CPU support using passed in Serial object
- Simple to read code
- Only MCP39F511 related code without any other peripherials

Please see an alternative / more-complex Arduino library [here](https://github.com/MatteoDestro/Arduino_MCP39F511)
